Controle de LEDs com Arduino

Este projeto foi desenvolvido no Tinkercad para simular um semáforo simples com Arduino e três LEDs: vermelho, amareloe verde. Os LEDs acendem em sequência, representando as fases de um semáforo real: "pare", "atenção" e "siga". É um projeto ideal para iniciantes, pois ensina conceitos básicos de eletrônica, controle de saídas digitais e lógica sequencial com Arduino.

Criando a interface de uma rede social

Pet Friends é uma rede social pet-friendly desenvolvida no Figma, idealizada para conectar tutores e amantes de animais. A interface foi projetada com foco na usabilidade, oferecendo uma navegação intuitiva e visual acolhedor. O app permite criar perfis de pets, compartilhar momentos, interagir com outros usuários e acessar serviços voltados ao bem-estar animal. Um espaço digital feito para celebrar a amizade entre humanos e seus pets.
